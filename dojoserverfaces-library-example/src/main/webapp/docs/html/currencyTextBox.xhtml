currency

  • Description: The [ISO4217](http://en.wikipedia.org/wiki/ISO_4217) currency code, a three letter sequence like "USD"
  • Type: java.lang.String
  • Required? false

disabled

  • Description: Whether or not this component is disabled.
  • Type: java.lang.Boolean
  • Required? false

fractional

  • Description: where places are implied by pattern or explicit 'places' parameter, whether to include the fractional portion.
  • Type: java.lang.Boolean
  • Required? false

locale

  • Description: override the locale on this widget only, choosing from djConfig.extraLocale
  • Type: java.lang.String
  • Required? false

maxLength

  • Description: The maximum length of input to be allowed by the client.
  • Type: java.lang.Integer
  • Required? false

pattern

  • Description: override localized convention with this pattern. As a result, all users will see the same behavior, regardless of locale, and your application may not be globalized. See http;//www.unicode.org/reports/tr35/#Number_Format_Patterns.
  • Type: java.lang.String
  • Required? false

placeHolder

  • Description: Help text displayed inside the input field.
  • Type: java.lang.String
  • Required? false

places

  • Description: number of decimal places to accept.
  • Type: java.lang.String
  • Required? false

rangeMessage

  • Description: The message to display if the value is not within the defined constraints
  • Type: java.lang.String
  • Required? false

readOnly

  • Description: Whether or not this component is read-only.
  • Type: java.lang.Boolean
  • Required? false

selectOnClick

  • Description: Indication that all text should be selected when field receives focus with mouse
  • Type: java.lang.Boolean
  • Required? false

strict

  • Description: strict parsing, false by default. When strict mode is false, certain allowances are made to be more tolerant of user input, such as 'am' instead of 'a.m.', some white space may be optional, etc.
  • Type: java.lang.String
  • Required? false

style

  • Description: The CSS style to associate with the rendered HTML element.
  • Type: java.lang.String
  • Required? false

styleClass

  • Description: The CSS class to associate with the rendered HTML element.
  • Type: java.lang.String
  • Required? false

symbol

  • Description: (currency only) override currency symbol. Normally, will be looked up in localized table of supported currencies (dojo.cldr) 3-letter ISO 4217 currency code will be used if not found.
  • Type: java.lang.String
  • Required? false

tabIndex

  • Description: The tab index for this component.
  • Type: java.lang.Integer
  • Required? false

trim

  • Description: Indication of whether or not leading and trailing whitespace are trimmed from the input field on the client.
  • Type: java.lang.Boolean
  • Required? false

type

  • Description: choose a format type based on the locale from the following; decimal, scientific (not yet supported), percent, currency. decimal by default.
  • Type: java.lang.String
  • Required? false

value

  • Description: The value, a number, for this component.
  • Type: java.lang.Number
  • Required? false
