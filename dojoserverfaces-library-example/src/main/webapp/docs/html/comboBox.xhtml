autoComplete

  • Description: If user types in a partial string, and then tab out of the field, automatically copy the first entry displayed in the drop down list to the field

  • Required? false

disabled

  • Required? false

hasDownArrow

  • Description: Set the textbox to have a down arrow button, to display the drop down list. Defaults to true.
  • Type: java.lang.Boolean
  • Required? false

highlightMatch

ignoreCase

maxLength

placeHolder

readOnly

  • Required? false

searchDelay

  • Type: java.lang.Integer
  • Required? false

selectOnClick
